# Report Contract

Deliverables (always):
- decision_brief.<lang>.md/json
- full_review.<lang>.md/json
- diagnosis_report.<lang>.md/json
- rebuttal.<lang>.md/json
- claim_discovery.json
- claim_evidence_matrix.json
- remediation_plan.json
- venue_profile_used.json
- skill_flow_used.json
- runtime_context.json

Deliverables (optional):
- decision_brief.<lang>.pdf
- full_review.<lang>.pdf
- diagnosis_report.<lang>.pdf
- rebuttal.<lang>.pdf

Optional PDF export is enabled only when `options.always_export_pdf=true`.

Language modes:
- en: English outputs only.
- en_zh: English and Chinese outputs.
